# Sales Performance Dashboard - Power BI

## 📌 Overview
This project is an interactive Sales Performance Dashboard created using Microsoft Power BI. It provides insights into sales performance through KPIs, charts, and filters, helping users analyze sales trends, product performance, customer information, and city-wise sales.

---

## 🚀 Features

- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value KPI
- Monthly Sales Trend
- Sales by Category
- Top Products by Sales
- Sales by City
- Quantity Sold by Product
- Average Sales by City
- Customer Details Table
- Interactive Slicers
  - Category
  - City
  - Gender

---

## 🛠 Tools Used

- Microsoft Power BI Desktop
- Microsoft Excel
- Power Query
